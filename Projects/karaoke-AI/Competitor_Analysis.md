## 国内競合

### 1. DAM（第一興商） — `clubdam.com`

**強み**
- 約 16万曲のデータベース、カラオケ BOX との連携が強固
- 精密採点 RB（リアルバトル）でリアルタイムランキング機能
- `clubdam.com/karaokesearch/` で無料 WEB 曲検索
- 楽曲リクエスト機能あり（`clubdam.com/karaokerequest/`）
- スマホアプリ「カラオケ@DAM」はユーザー評価が高い

**弱み**
- 曲数は JOYSOUND より少ない
- アニメ・ボカロジャンルが手薄

**ビジネスモデル**
- カラオケ BOX との月額ライセンス料
- アプリ課金（歌い放題プラン）

---

### 2. JOYSOUND（エクシング） — `joysound.com`

**強み**
- 曲数 **No.1**（19万曲以上）、毎月 700曲追加
- 分析採点：こぶし・しゃくり・ビブラートを個別判定
- WEB 曲検索 + 歌詞検索が充実（`joysound.com/web/search`）
- 2025年〜「**全力入曲**」プロジェクト：ユーザーリクエストで未配信曲を追加する差別化施策
- 「ジョイサウンドカフェ」でブラウザ上でリアルタイム曲検索

**弱み**
- アプリよりカラオケ BOX 側コンテンツに注力している印象
- 採点の評価軸が複雑でユーザーに伝わりにくい

**ビジネスモデル**
- カラオケ BOX ライセンス + アプリサブスク
- 「全力入曲」でユーザーエンゲージメント向上 → 配信権交渉コスト削減

## 海外・グローバル競合

### 3. Smule — `smule.com`

**強み**
- 曲数 **1,000万曲以上**（世界最大級）
- ピッチガイド、ボイストレーニング機能、デュエット録音
- SNS 共有 + コミュニティ（フォロー・コメント）が強い
- 「Reverse Singing」など独自機能

**弱み**
- 日本語楽曲の比率が低い
- 月額課金が高め

**ビジネスモデル**
- サブスクリプション（月額 / 年額）
- 市場シェア：Smule + WeSing で **52%** を占有

---

### 4. WeSing（Tencent） — グローバル

**強み**
- Tencent 資本でアジア圏に強い
- セクション歌唱（曲の一部だけ歌う）
- デュエット・コラボ機能が充実

**弱み**
- 中国 UI/UX が日本ユーザーに馴染みにくい
- 日本語楽曲の検索精度が低い

---

### 5. StarMaker

**強み**
- 音声エフェクト・フィルターが豊富
- ゲーム機能（Ludo, Draw & Guess）でエンタメ化
- SNS 動画として投稿しやすい UI

**弱み**
- 曲検索 UX よりも「映え」特化
- カラオケ曲数は Smule より少ない

---

### 6. KaraFun

**強み**
- 「Netflix of Karaoke」と称されるサブスク型
- 高品質音源、オフライン再生対応
- ホームユーザー向けに使いやすい UI

**弱み**
- 日本語楽曲が少ない
- 企業（KJ）向けに特化しており、ソーシャル機能ゼロ

---

### 7. Singa

**強み**
- 「Spotify of Karaoke」ポジション
- ヨーロッパで急成長、UX がモダン
- バー・会場向け B2B プランあり

**弱み**
- アジア・日本市場への展開は限定的

## 市場のギャップ（差別化チャンス）

1. **未配信曲の可視化**  
   「この曲はどこのカラオケにある？」を横断検索できるサービスがない。DAM / JOYSOUND をまたいだ配信状況の一元検索はブルーオーシャン。

2. **曲発見 UX の貧弱さ**  
   既存サービスは「知っている曲を探す」前提。「今日の気分・シーン・一緒に来た人の好みから探す」レコメンド機能はほぼ未開拓。

3. **SNS トレンド連動**  
   TikTok / Reels でバズった曲が「どのカラオケで入っているか」をリアルタイムで把握できるサービスがない。

4. **グループ選曲の非効率**  
   複数人で来た場合の合意形成ツール（みんなの候補をまとめる、投票する）が存在しない。

5. **AI レコメンド**  
   歌唱履歴・採点結果・ユーザー属性からのパーソナライズド推薦は、国内勢には未実装。
